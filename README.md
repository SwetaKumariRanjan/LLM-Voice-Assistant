# LLM-Voice-Assistant
AI Voice Assistant built with Python using SpeechRecognition and pyttsx3, featuring voice commands, web search, YouTube playback, desktop automation, and extensible AI capabilities.

A simple AI Voice Assistant built with Python that listens to voice commands and performs tasks such as opening websites, launching applications, searching Google, playing YouTube videos, and providing the current time and date.

**Features**
Speech Recognition
Text-to-Speech Responses
Open Google and YouTube
Open Notepad and Calculator
Search Google
Play Songs on YouTube
Tell Current Time and Date
Voice Command Support

**Installation**
git clone https://github.com/your-username/llm-voice-assistant.git
cd llm-voice-assistant
pip install -r requirements.txt

**Run**
python assistant.py

**Example Commands**
Assistant hello
Assistant open google
Assistant open youtube
Assistant search Python tutorials
Assistant play believer song
Assistant what is the time
Assistant what is today's date
Assistant exit
